## Installation & Usage

### Composer

To install the bindings via [Composer](http://getcomposer.org/), add the following to `composer.json`:

```json
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/elestyle/elepay-php-sdk.git"
    }
  ],
  "require": {
    "elestyle/elepay-php-sdk": "*@dev"
  }
}
```

Then run `composer install`
